How to work through workflow using Windows:

Setting Up A Repository:

To set up a repository on github follow these next steps:
  1. Hit the "+" button on the top right corner.
  2. Click New Repository.
  3. Name what you would like your repository to be called, and add a description if you would like one.
  4. Click the Read Me box.
  5. Click the Create box.
  6. Next, select the Settings box and scroll all the way to the bottom.
  7. Find the Source button, and select Master Branch, then hit Save.
  8. Now, click on the File tab, then New File.
  9. Name your new file index.html .
  10. Finally, click the Commit tab.
